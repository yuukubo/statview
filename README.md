目標は、  
自分の状態をエクセル更新のような感覚で変更可能。  
座席の背景色で状態が分かる。  
座席を編集、島の作成が出来る。  
マウスオーバーで自由入力のコメントがポップアップする。  
出来ないけど既存のスケジュール連動出来たら二重入力を排除出来る。  
入力対象が複数になるのは可能なら避けたいところ。
